# Spotnet Watch

Spotnet Watch is een programma om automatisch de Spotnet‑database te monitoren naar spots die overeenkomen met de door jou ingestelde zoekcriteria.  
Vergelijkbaar met het oude **Watchdog** dat ooit voor FTD is gemaakt.

## Status
**Beta** – het programma werkt, maar er zijn nog enkele verbeterpunten.

## Installatie
1. Download het zip‑bestand.  
2. Pak het zip‑bestand uit naar een gewenste map.  
3. Dubbelklik op `SpotnetWatch.exe`.

> **Vereiste:** .NET Framework **4.8** moet geïnstalleerd zijn voordat Spotnet Watch kan worden gestart.

## Gebruik
1. Start `SpotnetWatch.exe`.  
2. Stel je zoekcriteria in via het instellingen‑menu.  
3. Het programma controleert continu de Spotnet‑database en meldt matches.
